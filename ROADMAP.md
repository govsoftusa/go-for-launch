# Roadmap

## Current

- Complete Webflow-to-Astro migration workflow, proven in production.
- WordPress-to-Astro process, proven in production and documented in a case study.
- Platform-neutral migration framework. Guidance for platforms other than Webflow and WordPress is alpha and unproven in production.
- Mobile Safari and WebKit playbook.
- Testing and production-release checklist.
- Route, gap-analysis, and acceptance templates.
- Sanitized implementation case studies.

## Next

- Add a reusable route-audit Playwright package.
- Add a visual-reference capture helper.
- Add a platform-neutral asset-manifest schema.
- Add synthetic Webflow export fixtures.
- Add a sample Astro content importer.
- Add WordPress REST API migration guidance.
- Graduate the alpha Squarespace, Wix, Drupal, static HTML, and custom CMS guidance with production evidence.
- Add redirect-map validation examples.
- Add accessibility test examples using current maintained tooling.
- Add CI examples for Chromium, WebKit, link checks, and Markdown validation.

## Contribution Ideas

- Document a platform-specific export limitation.
- Contribute a migration adapter contract.
- Add a browser regression pattern.
- Add a case study with before and after evidence.
- Add a performance experiment, including unsuccessful approaches.
- Improve templates for large multilingual or commerce sites.
